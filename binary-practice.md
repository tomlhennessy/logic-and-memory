# Binary Practice

* Binary to base 10

1. 0b1010 => 10
2. 0b0011 => 3


* Binary to hexadecimal

1. 0b1010 => 0xA
2. 0b0011 => 0x3


* Hexadecimal to base 10

1. 0xa1 => 161
2. 0xff => 255


* Hexadecimal to binary

1. 0xa1 => 0b10100001
2. 0xff => 0b11111111


* Base 10 to binary

1. 8 => 0b1000
2. 24 => 0b11000
3. 255 => 0b11111111


* Base 10 to hexadecimal

1. 8 => 0x8
2. 24 => 0x18
3. 255 => 0xFF


* Base 10 to ASCII

1. 65 => A
2. 66 => B
3. 97 => a
4. 98 => b
